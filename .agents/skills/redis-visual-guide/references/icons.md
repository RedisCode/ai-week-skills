# Icons

Redis icons are organized into four categories, each with a distinct purpose and usage context. All icon assets are SVG files located in `assets/` and are available in four color variants: **Duotone**, **Hyper**, **Midnight**, and **White**.

## File Naming Convention

Icons follow the pattern:

```
assets/<Category>/SVG/64x64/<Variant>/icon-<name>-64-<variant>.svg
```

For example: `assets/Generic Marketing Icons/SVG/64x64/Duotone/icon-AI-64-duotone.svg`

Variant values are lowercase in filenames: `duotone`, `hyper`, `midnight`, `white`.

## Color Variants

| Variant  | Description                                     |
| -------- | ----------------------------------------------- |
| Duotone  | Two-tone style using brand colors               |
| Hyper    | Uses the Hyper (red) color scale                |
| Midnight | Uses the Midnight dark color for dark contexts  |
| White    | White variant for use on dark backgrounds       |
